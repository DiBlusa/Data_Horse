---

## ✨ Overview

**Data Horse** is a personal project designed to help classify and organize **Minecraft horse** information in a practical, data-driven way.  
The core goal is to build a reliable foundation for horse categorization, analysis, and decision support.

---

## 🎯 Project Goals

- Classify Minecraft horses using structured data.
- Keep records organized and easy to expand.
- Improve consistency in how horse profiles are evaluated.

---

## 🧬 Future Vision

This project is being built with growth in mind.  
One of the next major updates will be:

- **Genealogical Tree (Pedigree) support**
- Lineage tracking and family relationship visualization
- Better context for breeding and long-term performance analysis

---

## ⚙️ Current Structure

The repository currently includes:

- Core Python scripts for project logic
- Local database for horse data storage
- Execution helpers for quick local runs

---

## 🚀 Getting Started

1. Clone this repository.
2. Create/activate your Python virtual environment.
3. Run the main script and start managing horse data.

Example:

```bash
python main.py
```
